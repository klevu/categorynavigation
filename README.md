# Category Navigation
This is a sub package to the klevu-smart-search-M2 repository. It is for
showing Klevu based template in category navigation pages.

<hr />
<h2>Compatible with</h2>
<li>CE 2.0.0 to 2.3.0</li>
<li>EE 2.0.0 to 2.3.0</li>
